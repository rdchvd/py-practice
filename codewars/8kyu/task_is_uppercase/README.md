# Task: is_uppercase

## Description
Create a method to see whether the string is ALL CAPS.

### Example:
- Input: `"c"`
- Output: `False`
- Input: `"C"`
- Output: `True`

- Input: `"hello I AM DONALD"`
- Output: `False`
- Input: `"HELLO I AM DONALD"`
- Output: `True`

### Solution
The function uses method `.upper()` to compare the string with its uppercase equivalent.
